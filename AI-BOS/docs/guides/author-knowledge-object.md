# How to Author a Knowledge Object

## Steps

1. Copy `TP-KNW-001` (Knowledge Object Template).
2. Fill required placeholders (id, title, type, category, domain, version, status, owner, visibility, audience, priority, tags).
3. Write all nine mandatory body sections.
4. Validate against `ST-KNW-001`.
5. Register the object in `knowledge-registry/registry.json` and the type-specific index.
6. Record evidence in `tasks.md`.

## Presents Knowledge Objects

- ST-KNW-001 — Knowledge Object Standard
- TP-KNW-001 — Knowledge Object Template
