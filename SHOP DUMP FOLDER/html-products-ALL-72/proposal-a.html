01 // OVERVIEW

The Opportunity

NovaTech Properties requires a modern property listing and management platform to replace their current static website. This proposal outlines our approach to building a scalable, performant web application that serves both end users (property seekers) and administrators (internal team).

Based on our discovery call and audit of your current setup, we've identified the primary gaps: no search functionality, no agent portal, mobile experience scoring 28/100 on Google PageSpeed, and zero CRM integration. This proposal addresses all four.

02 // SCOPE OF WORK

What We're Building

  • 🏠
    Property Listing Portal

    Full-featured listing pages with advanced search, filter by location/price/type, map integration, and photo galleries. SEO-optimised for each listing.

  • 🔑
    Agent & Admin Dashboard

    Internal CMS for the NovaTech team to create, edit, and manage listings without developer help. Role-based access control.

  • 📱
    Mobile-First Responsive Design

    Rebuilt from the ground up for mobile. Target: 90+ PageSpeed score on mobile and desktop.

  • 🔗
    CRM & Lead Integration

    Connect enquiry forms directly to HubSpot CRM. Automated lead assignment to agents. Email notification workflows.

  • ☁️
    Hosting & DevOps Setup

    AWS deployment with staging environment, automated deployments via GitHub Actions, and monitoring via Datadog.
