# Technical SEO Agent Skills

Document status: framework module

## Purpose

Define the competencies the Technical SEO Agent must apply when reviewing, auditing, or planning technical SEO work.

## Technical SEO

- Google Search Essentials.
- Crawl optimization and crawl budget review.
- Index management.
- Canonical strategy.
- Robots.txt and robots meta.
- XML sitemaps and sitemap indexes.
- Structured data and Schema.org.
- Internal linking and navigation.
- Pagination, faceted navigation, and search pages.
- International SEO and hreflang.
- Local SEO and LocalBusiness schema.

## Next.js And Modern Web Architecture

- App Router.
- Metadata API.
- Server Components and Client Components.
- Route Handlers and Server Actions.
- Static generation, SSR, ISR, streaming, and Partial Prerendering.
- Edge Runtime and caching behavior.
- Rendering visibility for crawlers.

## Performance Engineering

- Core Web Vitals: LCP, INP, CLS.
- Bundle optimization and code splitting.
- Image optimization and responsive media.
- Lazy loading and priority loading.
- Critical rendering path.
- Browser caching, CDN caching, and edge caching.
- API and database performance for SEO-critical routes.

## Security And HTTP

- HTTPS and TLS.
- CSP, HSTS, X-Content-Type-Options, Referrer-Policy, Permissions-Policy.
- XSS, CSRF, CORS, authentication, authorization.
- Status codes, redirects, soft 404s, error handling, compression, and cache headers.

## DevOps And Observability

- CI/CD release gates.
- GitHub Actions, Vercel, Cloudflare, Docker, and environment management concepts.
- Rollbacks and migration launch plans.
- Monitoring, logging, alerting, and crawler log analysis.

## Analytics And Webmaster Tools

- GA4.
- Google Search Console.
- Bing Webmaster Tools.
- Microsoft Clarity.
- PostHog.
- Real-user monitoring and synthetic performance monitoring.

## Communication Skills

- Executive summaries.
- Severity-scored findings.
- Implementation briefs.
- AI-executable tasks.
- Evidence-based recommendations.
