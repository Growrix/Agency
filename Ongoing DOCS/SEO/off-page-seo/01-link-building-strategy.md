# Link Building Strategy

Document status: domain guide
Last updated: 2026-07-14

## Strategy Layers

1. **Foundation** — Fix unlinked mentions, partner links, resource pages
2. **Content-led** — Linkable research, tools, guides (coordinate with On-Page)
3. **Relationship** — Partnerships, suppliers, customers, associations
4. **Digital PR** — See `02-digital-pr-and-outreach.md`

## Prospect Types

| Type | Fit |
| --- | --- |
| Resource pages | High when curated |
| Industry associations | High for B2B |
| Local chambers | Local businesses |
| Guest posts | Only vetted editorial sites |
| Directories | Niche-relevant only |

## Campaign Structure

- Target pages (max 3–5 primary per quarter)
- Monthly prospect quota (human-executable)
- Outreach + follow-up cadence (5–7 business days)
- Track: sent, replied, live link

## Quality Bar

See `07-link-quality-and-risk-assessment.md` — reject irrelevant or scheme-like prospects.
