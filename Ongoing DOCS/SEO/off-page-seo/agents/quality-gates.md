# Off-Page SEO Quality Gates

Document status: active
Last updated: 2026-07-14

## Blocking (escalate)

- Active toxic link campaign or PBN exposure (recommend disavow + cleanup plan)
- NAP inconsistencies on major directories (local)
- GBP suspended or duplicate listings unresolved
- Outreach plan violates handbook (paid links, link schemes)

## Non-Blocking

- Low-authority but relevant directory opportunities
- Additional PR angles
- Social profile completeness

## Link Quality Minimum

Prospective links should meet:

- Topical relevance to target page
- Editorial context (not sitewide footer spam)
- Real traffic or legitimate niche authority
- No obvious link scheme patterns

## Severity Scale

Critical / High / Medium / Low / Informational — same as other SEO agents.
