# Off-Page SEO Operating Modes

Document status: active
Last updated: 2026-07-14

## Mode Selection

| Mode | Use when |
| --- | --- |
| **Audit Mode** | Review existing backlink profile, citations, mentions |
| **Link Building Mode** | Prospecting and outreach campaigns |
| **Digital PR Mode** | News, data studies, story pitches |
| **Local SEO Mode** | Multi-location, GBP, local pack goals |

State selected mode before deep work if unspecified.

## Audit Mode Workflow

```text
Scope + business goals
→ collect backlink/citation data (user exports)
→ assess quality, relevance, risk, gaps vs competitors
→ NAP/GBP consistency check (local)
→ severity-scored report + roadmap
```

## Link Building Mode Workflow

```text
Define linkable assets and target pages
→ prospect criteria and list building methodology
→ outreach templates and follow-up cadence
→ success metrics and risk guardrails
```

## Digital PR Mode Workflow

```text
Identify newsworthy hooks (data, product, story)
→ media/journalist target list framework
→ pitch templates and embargo process
→ measurement (mentions, links, referral traffic)
```

## Local SEO Mode Workflow

```text
Location inventory
→ NAP audit across web
→ GBP optimization checklist
→ citation building plan + review strategy
→ hand off schema to Technical
```
