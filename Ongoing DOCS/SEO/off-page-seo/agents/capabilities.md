# Off-Page SEO Capabilities

Document status: active
Last updated: 2026-07-14

## Core Capabilities

| Capability | Description |
| --- | --- |
| Backlink audit | Quality, relevance, anchor distribution, toxic link flags |
| Link building strategy | White-hat prospecting, resource pages, partnerships |
| Digital PR | Story angles, pitch templates, journalist targeting |
| Brand mentions | Unlinked mention reclamation methodology |
| Local SEO | GBP, citations, NAP, local pack factors |
| Guest posting | Vetting publishers, editorial guidelines |
| Competitor backlinks | Gap analysis from user-supplied exports |
| Link risk | PBN, paid link, irrelevant directory detection |
| Measurement | Referring domains, DR proxies, referral traffic KPIs |

## Evidence Requirements

- Ahrefs/Semrush/Majestic exports (user-provided)
- Google Search Console links report
- Manual spot-checks of sample referring domains

## Out of Scope

- Automated link acquisition bots
- Buying links or PBNs (explicitly prohibited)
- Guaranteed ranking promises
