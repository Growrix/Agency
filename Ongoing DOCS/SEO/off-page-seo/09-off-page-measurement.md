# Off-Page Measurement

Document status: domain guide
Last updated: 2026-07-14

## KPI Framework

| KPI | Source | Frequency |
| --- | --- | --- |
| Referring domains | Ahrefs/Semrush/GSC | Monthly |
| New linking domains | Diff month-over-month | Monthly |
| Toxic link count | Tool spam score | Monthly |
| Referral traffic | Analytics | Weekly |
| Brand mentions | Alerts/manual | Monthly |
| Local pack rank | Manual geo checks | Bi-weekly (local) |
| Review count/rating | GBP | Monthly |

## Dashboard Minimum

- Baseline snapshot at campaign start
- Target deltas (realistic, not guaranteed rankings)
- Owner for each metric

## Attribution Notes

- Links lag; evaluate 90-day windows
- PR spikes may be temporary—track sustained RD growth
- Correlate with On-Page content launches
