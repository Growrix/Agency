# Off-Page SEO Pre-Launch Checklist

Document status: active checklist
Last updated: 2026-07-14

## Link Foundation

- [ ] Target pages identified with On-Page intent alignment
- [ ] Linkable assets ready (research, tools, guides, press kit)
- [ ] Baseline referring domain count recorded (if data available)

## Local (if applicable)

- [ ] NAP consistent on site and top citations
- [ ] Google Business Profile claimed and complete
- [ ] No duplicate GBP listings
- [ ] LocalBusiness schema plan → Technical handoff

## Brand & PR

- [ ] Press/contact page live
- [ ] Brand monitoring baseline (manual or tool)
- [ ] Social profiles claimed (consistent branding)

## Risk

- [ ] No active link scheme exposure
- [ ] Outreach templates reviewed for compliance with rules

## Post-Launch

- [ ] GSC links monitored monthly
- [ ] Referral traffic segment in analytics
- [ ] Citation audit scheduled (local)
