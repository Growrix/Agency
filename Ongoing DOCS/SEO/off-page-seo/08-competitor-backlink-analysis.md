# Competitor Backlink Analysis

Document status: domain guide
Last updated: 2026-07-14

## Process

1. Define 3–5 SERP competitors (not just business competitors)
2. Import backlink exports (Ahrefs/Semrush — user supplied)
3. Compare referring domain counts and top linked pages
4. Identify gap categories: directories, PR, resources, guest posts
5. Prioritize replicable (white-hat) opportunities

## Replication Rules

- Do not copy spam tactics competitors use
- Focus on gaps you can earn ethically
- Align target URLs with On-Page intent map

## Anti-Hallucination

Competitor DR/RD counts require user exports—do not invent.
