# Local SEO and Citations

Document status: domain guide
Last updated: 2026-07-14

## Core Elements

- **NAP** — Name, Address, Phone identical everywhere
- **GBP** — Categories, hours, photos, services, posts, Q&A
- **Citations** — Consistent listings on relevant directories
- **Reviews** — Volume, velocity, response rate (policy-compliant requests)

## Citation Priority

1. Data aggregators (locale-dependent)
2. Industry-specific directories
3. Local chambers and associations
4. General directories (only if reputable and relevant)

## GBP Checklist

- [ ] Primary category accurate
- [ ] Service areas defined
- [ ] Website URL correct
- [ ] UTM not in GBP website field
- [ ] Regular posts (where maintained)
