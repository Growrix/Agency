# Brand Mentions and Reputation

Document status: domain guide
Last updated: 2026-07-14

## Mention Types

| Type | Action |
| --- | --- |
| Unlinked brand mention | Request editorial link to relevant page |
| Incorrect NAP | Request correction |
| Negative review | Respond per brand policy; escalate crises |
| Duplicate listings | Merge/remove (local) |

## Monitoring (Manual Framework)

- Google Alerts for brand + key products
- Periodic `brand name` SERP review
- Social listening (user tools)

## Reclamation Email Template Structure

1. Thank them for the mention
2. Cite specific URL and context
3. Suggest relevant link target
4. No pressure or incentives for links

## E-E-A-T Connection

Authoritative mentions support trust; coordinate with On-Page for about/press pages.
