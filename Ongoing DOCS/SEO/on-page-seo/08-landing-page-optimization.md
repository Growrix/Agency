# Landing Page Optimization

Document status: domain guide
Last updated: 2026-07-14

## Landing Page Types

| Type | SEO focus |
| --- | --- |
| Service/product | Commercial/transactional intent, clear H1, proof, CTA |
| Lead gen | Single intent; avoid blog-style dilution |
| Campaign | Often noindex unless evergreen (Technical) |
| Local service area | Geo modifiers + NAP (Off-Page for citations) |

## Above-the-Fold

- H1 matches ad/SERP promise (message match)
- Primary value prop visible without scroll
- Trust signals (logos, reviews, guarantees)

## On-Page SEO Elements

- Unique title/meta for each landing variant
- Schema recommendations (FAQ, Service) → Technical
- Internal links to supporting proof (case studies, pricing)

## Conversion + SEO Balance

Do not sacrifice clarity for keywords. One primary CTA per page.

## Testing

- A/B headlines affect CTR from ads; SEO titles may differ—document both if needed.

## Handoff

Copy and section structure → content strategist; forms/tracking → SaaS dev.
