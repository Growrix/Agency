# Title and Meta Description Optimization

Document status: domain guide
Last updated: 2026-07-14

## Title Tags

- **Length:** ~50–60 characters display target (avoid truncation on mobile).
- **Structure:** Primary topic | Secondary context | Brand (when space allows).
- **Uniqueness:** Every indexable URL gets a unique title.
- **Intent match:** Title must match SERP expectation for primary query.

## Meta Descriptions

- **Length:** ~150–155 characters target.
- **Purpose:** CTR improvement; not a direct ranking factor but affects clicks.
- **Include:** Value prop, differentiator, soft CTA when appropriate.
- **Avoid:** Keyword stuffing, duplicate boilerplate across pages.

## OG / Social Copy

Recommend `og:title` and `og:description` when they should differ from SERP snippets (Technical implements).

## Audit Checklist

- [ ] Duplicate titles across indexable pages
- [ ] Missing or generic titles ("Home", "Services")
- [ ] Titles misaligned with H1 or body topic
- [ ] Meta missing or duplicated at scale
