# On-Page SEO Operating Modes

Document status: active
Last updated: 2026-07-14

## Mode Selection

| Mode | Use when |
| --- | --- |
| **Audit Mode** | Existing pages underperform; CTR low; content misaligned with intent |
| **Build Mode** | New landing pages, service pages, route groups, site sections |
| **Content Strategy Mode** | Editorial planning, topic clusters, competitive content gaps |

State the selected mode before deep work if the user did not specify.

## Audit Mode Workflow

```text
Define page scope and business goals
→ map primary/secondary intent per URL
→ audit title, meta, H1, headings, body depth, E-E-A-T
→ audit internal links in/out
→ score gaps vs SERP competitors (user-supplied or stated assumptions)
→ prioritize findings by impact
→ produce audit report + implementation brief
```

## Build Mode Workflow

```text
Business goals + audience
→ keyword research and intent mapping
→ URL/page type recommendations (hand off URL rules to Technical)
→ title/meta/H1 recommendations per page
→ content outline and word-count targets
→ internal link plan
→ brief for content strategist and technical implementer
```

## Content Strategy Mode Workflow

```text
Market and competitor landscape
→ topic cluster map
→ pillar + supporting content plan
→ E-E-A-T and author strategy
→ measurement plan (GSC, analytics)
→ phased roadmap
```

## Deliverables by Mode

| Mode | Primary outputs |
| --- | --- |
| Audit | Severity-scored on-page audit, quick wins, content refresh list |
| Build | Keyword map, page briefs, recommended titles/metas, link plan |
| Content Strategy | Cluster map, editorial calendar, gap analysis, KPI framework |
