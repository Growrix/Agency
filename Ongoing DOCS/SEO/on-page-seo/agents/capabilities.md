# On-Page SEO Capabilities

Document status: active
Last updated: 2026-07-14

## Core Capabilities

| Capability | Description |
| --- | --- |
| Keyword research | Primary/secondary keywords, intent labels, difficulty proxies |
| Intent mapping | Informational, commercial, transactional, navigational per URL |
| Title & meta strategy | Recommended titles, descriptions, OG copy (not HTML implementation) |
| Heading architecture | H1 uniqueness, H2–H6 outline, semantic hierarchy |
| Content depth analysis | Thin content detection, SERP parity recommendations |
| E-E-A-T planning | Author bios, credentials, citations, trust signals |
| Internal linking | Hub/spoke plans, anchor text guidance, orphan page fixes |
| Landing page optimization | Conversion + SEO alignment for key money pages |
| Content hubs | Pillar/cluster design for blog and resource sections |
| Competitor content gaps | Topics competitors rank for that you lack (manual/tool input) |

## Evidence Sources

- Page content (files, CMS export, live URL)
- User-supplied GSC queries/impressions
- User-supplied Ahrefs/Semrush content exports
- SERP manual review (labeled as observation)

## Not Capable Without External Input

- Live rank tracking automation
- Automated SERP scraping at scale
- CMS publishing (produces briefs only)
