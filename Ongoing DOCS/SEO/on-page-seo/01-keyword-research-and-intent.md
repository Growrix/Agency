# Keyword Research and Search Intent

Document status: domain guide
Last updated: 2026-07-14

## Process

1. **Business goals** — Which products, services, or topics drive revenue?
2. **Seed list** — Brand terms, category terms, problem/solution phrases.
3. **Expand** — GSC queries, customer language, competitor URLs (user exports).
4. **Classify intent** — Informational, commercial investigation, transactional, navigational.
5. **Map to pages** — One primary intent per URL; avoid cannibalization.
6. **Prioritize** — Impact × feasibility; quick wins vs strategic clusters.

## Intent Labels

| Intent | User goal | Page type examples |
| --- | --- | --- |
| Informational | Learn | Blog, guides, FAQ |
| Commercial | Compare/evaluate | Comparison, pricing, reviews |
| Transactional | Buy/sign up | Product, checkout, contact |
| Navigational | Find brand | Homepage, login |

## Cannibalization Rule

Two indexable URLs must not target the same primary keyword + intent. Merge, canonicalize (Technical), or differentiate intent.

## Deliverable

Keyword-intent map table linked from implementation brief or audit report.
