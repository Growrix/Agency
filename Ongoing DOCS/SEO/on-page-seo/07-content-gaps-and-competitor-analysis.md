# Content Gaps and Competitor Analysis

Document status: domain guide
Last updated: 2026-07-14

## Process

1. Define competitor set (direct + SERP competitors for target keywords)
2. Collect user-supplied Ahrefs/Semrush content gap export OR manual URL inventory
3. Map topics competitors rank for that you lack
4. Score gaps: traffic potential, business fit, effort
5. Prioritize into content roadmap

## Gap Categories

| Type | Action |
| --- | --- |
| Missing topic | New page or article in cluster |
| Thin existing page | Expand or merge |
| Wrong intent | Retarget or redirect (Technical) |
| Outdated | Refresh with new E-E-A-T signals |

## Competitor Page Teardown

For each priority competitor URL document:

- Primary intent and keyword
- Title, H1, approximate depth
- Unique sections you lack
- Internal links and CTAs

## Anti-Hallucination

Do not invent competitor traffic numbers. Use exports or qualitative assessment.

## Deliverable

Gap analysis table + prioritized content backlog in Content Strategy Mode.
