# Internal Linking

Document status: domain guide
Last updated: 2026-07-14

## Goals

- Distribute authority to money pages and hubs
- Help crawlers discover indexable content
- Guide users through the conversion journey

## Patterns

| Pattern | Use |
| --- | --- |
| Hub → spoke | Pillar page links to cluster articles |
| Contextual | In-body links with descriptive anchors |
| Nav/footer | Stable paths to services, pricing, contact |
| Breadcrumbs | Hierarchy clarity (Technical implements) |

## Anchor Text

- Descriptive, varied, natural—avoid exact-match spam.
- Match target page intent.
- Do not use "click here" for SEO-critical links.

## Audit Steps

1. List indexable URLs in scope
2. Map inbound internal link count (crawl export or manual)
3. Flag orphans (zero internal inlinks)
4. Recommend links from high-authority templates (homepage, nav hubs)

## Cannibalization

When two pages compete, use internal links to reinforce the **preferred** URL for a topic.

## Handoff
