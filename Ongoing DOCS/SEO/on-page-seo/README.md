## Purpose

Single source of truth for **on-page SEO**: keyword research, search intent, title/meta strategy, heading structure, content strategy, E-E-A-T, internal linking, landing pages, and content hubs.

## Scope

**In scope:** Keyword research, intent mapping, title/meta copy recommendations, H1–H6 strategy, content briefs, E-E-A-T signals, internal linking plans, content gap analysis, landing-page optimization, blog/hub architecture.

**Out of scope (hand off):**

| Topic | Hand off to |
| --- | --- |
| Metadata implementation (Next.js, HTML `<head>`) | `Technical_SEO_expert` |
| Schema plumbing, canonicals, robots, sitemaps | `Technical_SEO_expert` |
| Final brand-voice copywriting | `@frontend-content-strategist` |
| Backlinks, PR, citations | `Off_Page_SEO_expert` |
| CWV, render performance | `Technical_SEO_expert` |
