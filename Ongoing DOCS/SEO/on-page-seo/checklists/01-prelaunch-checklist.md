# On-Page SEO Pre-Launch Checklist

Document status: active checklist
Last updated: 2026-07-14

## Intent & Keywords

- [ ] Primary keyword and intent documented per indexable URL
- [ ] Secondary keywords mapped to sections (H2/H3)
- [ ] No two money pages compete for the same primary intent

## Titles & Meta

- [ ] Unique title tag recommended per indexable page (≤60 chars target)
- [ ] Unique meta description per indexable page (≤155 chars target)
- [ ] Title includes primary topic; brand suffix consistent
- [ ] Technical agent confirms metadata implemented (handoff complete)

## Headings & Content

- [ ] Single H1 per page aligned with intent
- [ ] Logical H2–H6 outline; no skipped levels for styling
- [ ] Content depth meets SERP parity for target intent
- [ ] E-E-A-T elements present on YMYL pages (author, sources, dates)

## Internal Linking

- [ ] Key pages linked from nav or hub pages
- [ ] Contextual internal links with descriptive anchors
- [ ] No orphan indexable pages in scope

## Media (content layer)

- [ ] Descriptive alt text recommendations for informative images
- [ ] File names descriptive where applicable (Technical validates delivery)

## Measurement

- [ ] GSC property ready for post-launch monitoring
- [ ] Baseline impressions/clicks recorded if available
