# Composition Library

The `bp-05-homepage-composition-architect` chooses a **narrative composition** that dictates section order and user journey. 

**Never use `Hero → Features → Testimonials → CTA` without explicit justification.** That default is allowed only if the agent argues why it beats every alternative for this specific buyer.

Each composition lists: **Narrative logic**, **Typical section flow**, **Conversion mechanic**, **Best for**, **Buyer awareness fit**.

---

## 1. Storytelling
- **Narrative logic:** Take the visitor on an emotional arc — problem → tension → resolution → invitation.
- **Flow:** Atmospheric hero → the problem (status quo pain) → turning point → the solution as protagonist → proof → vision of life-after → CTA.
- **Conversion mechanic:** Emotional resonance + a single strong CTA at the climax.
- **Best for:** Mission-driven brands, wellness, nonprofits, premium lifestyle, founder-led.
- **Awareness fit:** Problem-aware → solution-aware.

## 2. Transformation
- **Narrative logic:** Before → After. Sell the gap and how to cross it.
- **Flow:** Hero promising the after-state → current pain → the bridge (your method) → before/after proof → step-by-step path → guarantee → CTA.
- **Conversion mechanic:** Visible transformation + risk reversal.
- **Best for:** Fitness, coaching, home renovation, med-spa, B2B "results" services.
- **Awareness fit:** Problem-aware → most-aware.

## 3. Marketplace
- **Narrative logic:** Two-sided value — show supply richness and demand ease.
- **Flow:** Search/discovery hero → category browse → how-it-works (both sides) → featured listings → trust/safety → reviews → dual CTA (buy/sell or browse/list).
- **Conversion mechanic:** Discovery + reduced friction to first action.
- **Best for:** Marketplaces, directories, booking platforms, real estate, multi-vendor.
- **Awareness fit:** Solution-aware → most-aware.

## 4. Authority
- **Narrative logic:** Establish dominance and credibility first; the buyer should feel they're choosing the expert.
- **Flow:** Confident hero (claim + proof) → credentials/awards → signature methodology → case results → media/logos → leadership → CTA to consult.
- **Conversion mechanic:** Trust stacking → consultation request.
- **Best for:** Law, finance, consulting, B2B enterprise, medical specialists, agencies.
- **Awareness fit:** Solution-aware → product-aware.

## 5. Interactive
- **Narrative logic:** Let the visitor *do* something to understand value (learn by playing).
- **Flow:** Interactive hero (tool/configurator) → live result → explanation of what they just saw → deeper features → proof → CTA tied to their input.
- **Conversion mechanic:** Engagement + personalized output as the lead trigger.
- **Best for:** Product configurators, calculators, design tools, fintech, SaaS with "aha" moments.
- **Awareness fit:** Solution-aware → most-aware.

## 6. Educational
- **Narrative logic:** Teach first, sell second. Build trust through generosity.
- **Flow:** Hero framing a question → key concepts → guide/framework → resources → proof of expertise → soft CTA (guide/newsletter) → hard CTA.
- **Conversion mechanic:** Value-first lead magnet → nurture.
- **Best for:** Complex/considered purchases, B2B, finance, health, high-ticket services.
- **Awareness fit:** Unaware → problem-aware.

## 7. Portfolio
- **Narrative logic:** The work *is* the pitch. Show, don't tell.
- **Flow:** Bold hero statement → curated case studies (visual-first) → process → capabilities → recognition → contact.
- **Conversion mechanic:** Craft demonstration → inquiry.
- **Best for:** Agencies, studios, photographers, architects, designers, freelancers.
- **Awareness fit:** Solution-aware → product-aware.

## 8. Membership
- **Narrative logic:** Sell belonging and ongoing value, not a one-time purchase.
- **Flow:** Community/identity hero → what you get (recurring value) → inside-the-membership preview → tiers/pricing → member proof → guarantee → join CTA.
- **Conversion mechanic:** Identity + recurring value + FOMO.
- **Best for:** Communities, subscriptions, SaaS plans, gyms, content platforms, clubs.
- **Awareness fit:** Product-aware → most-aware.

## 9. Lifestyle
- **Narrative logic:** Sell the world and identity around the product.
- **Flow:** Aspirational lifestyle hero → mood/world-building → product woven into life → values → social proof/UGC → shop/CTA.
- **Conversion mechanic:** Aspiration + identity alignment.
- **Best for:** DTC, fashion, hospitality, travel, food/beverage, premium consumer goods.
- **Awareness fit:** Unaware → solution-aware.

## 10. Product Discovery
- **Narrative logic:** Help visitors find the right product fast among many.
- **Flow:** Search/filter hero → categories → guided finder/quiz → featured/bestsellers → comparison → reviews → add-to-cart/CTA.
- **Conversion mechanic:** Guided selection → reduced choice paralysis.
- **Best for:** E-commerce with broad catalogs, electronics, beauty, multi-SKU brands.
- **Awareness fit:** Solution-aware → most-aware.

## Selection guidance for `bp-05`
1. Match composition to the dominant **buyer awareness level** from Stage 01.
2. Match to the **conversion goal** from Stage 01 / Stage 07 (lead-gen vs purchase vs membership).
3. Cross-check `_uniqueness-ledger.md` — same-industry projects should use different compositions.
4. In `05-homepage-composition.md`, list the exact section order top-to-bottom, the purpose of each section, and the narrative reason it sits there.
