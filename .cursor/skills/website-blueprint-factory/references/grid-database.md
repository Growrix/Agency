# Grid Database

The grid system is a uniqueness dimension. Same-industry projects should not share the same grid. Each entry lists: **Structure**, **Feel**, **Best for**, **Avoid for**, **Pairs with** (styles/heroes).

---

## G1 — Swiss Grid
- **Structure:** Strict, mathematical column grid (often 12-col) with a consistent baseline; flush-left alignment; modular spacing.
- **Feel:** Rational, ordered, objective, timeless.
- **Best for:** Editorial, design tools, B2B clarity, museums, documentation.
- **Avoid for:** Playful, organic, or chaotic brand expressions.
- **Pairs with:** Swiss/Minimalism/Monochromatic styles; Magazine/Typographic heroes.

## G2 — Bento Grid
- **Structure:** Grid of rounded cards of varied sizes (1x1, 2x1, 2x2) forming a balanced mosaic; each cell a self-contained unit.
- **Feel:** Modular, organized, modern, "keynote."
- **Best for:** Feature/product showcases, SaaS, hardware, portfolios, stat displays.
- **Avoid for:** Long-form narrative reading flows.
- **Pairs with:** Bento Interface/Glassmorphism/Futuristic; Bento/Dashboard heroes.

## G3 — Editorial Grid
- **Structure:** Magazine-style multi-column with intentional asymmetry, pull quotes, varied image sizes, generous margins.
- **Feel:** Curated, sophisticated, story-led.
- **Best for:** Luxury, fashion, publishing, hospitality, storytelling brands.
- **Avoid for:** Data-dense dashboards, utilitarian tools.
- **Pairs with:** Editorial/Hospitality/Dark Luxury styles; Magazine/Cinematic heroes.

## G4 — Asymmetrical Grid
- **Structure:** Deliberately off-balance placement, broken columns, overlapping elements, dynamic negative space.
- **Feel:** Energetic, creative, confident, contemporary.
- **Best for:** Agencies, creative studios, fashion, bold DTC, art.
- **Avoid for:** Trust-critical, conservative, or comparison-heavy contexts.
- **Pairs with:** Brutalism/Retro/Cyberpunk styles; Layered/Typographic heroes.

## G5 — Dashboard Grid
- **Structure:** Dense, panel-based layout (cards, charts, sidebars) with clear regions and functional zoning.
- **Feel:** Powerful, data-rich, professional.
- **Best for:** Analytics, fintech, B2B platforms, ops tools, security.
- **Avoid for:** Emotional lifestyle/luxury storytelling.
- **Pairs with:** Data-Centric/Futuristic styles; Dashboard/Command Center heroes.

## G6 — Masonry Grid
- **Structure:** Pinterest-style variable-height columns packed tightly; content-length-driven.
- **Feel:** Organic, abundant, browse-friendly.
- **Best for:** Galleries, portfolios, blogs, e-commerce discovery, UGC, photography.
- **Avoid for:** Strict comparison or step-by-step processes.
- **Pairs with:** Organic/Retro/Lifestyle styles; Search First/Mosaic heroes.

## G7 — Split Grid
- **Structure:** Page divided into strong halves/verticals (often sticky one side, scrolling other); diagonal or hard-edge splits.
- **Feel:** Dual, decisive, high-contrast.
- **Best for:** Dual-audience brands, before/after, product+form pairings, comparison.
- **Avoid for:** Content needing many equal columns.
- **Pairs with:** Dark Luxury/Skeuomorphic/Industrial styles; Split Screen/Before-After heroes.
